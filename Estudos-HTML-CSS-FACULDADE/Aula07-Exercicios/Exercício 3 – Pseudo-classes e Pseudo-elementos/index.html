Aroma di Roma

Na Aroma di Roma, a pizza é uma celebração da tradição e do sabor autêntico. Cada massa é fermentada lentamente, resultando em uma base leve e aerada com bordas crocantes. Nossas receitas, passadas de geração em geração, usam apenas os ingredientes mais frescos, como tomates San Marzano e queijo mozzarella de búfala, garantindo que cada fatia seja uma viagem direta aos sabores clássicos de Nápoles.
