# Exercício 1 — Guia de Viagem CSS

## Objetivo
Demonstrar os principais seletores CSS: seletor universal, de elemento, de classe, de ID e combinados.

## Como testar
1. Abra `index.html` no navegador.
2. Observe: o item com id `#destaque` está visualmente diferenciado (prioridade de ID).
3. Mude as classes `.praia`, `.cidade`, etc., nos `<li>` para ver as cores trocarem.

## O que praticar
- Alterar a ordem das regras no `style.css` para ver o efeito da cascade.
- Trocar `.card p` para `#destaque .desc` e observar aumento na especificidade.
- Remover o seletor universal `*` e testar como muda o tamanho/box-model.
