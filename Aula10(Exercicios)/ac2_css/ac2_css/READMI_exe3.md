# 💻 EXERCÍCIO 3: Projeto Intermediário - Review de Produto Responsivo

Este repositório contém a solução para o **Exercício 3**, focado na aplicação prática do **Modelo de Caixas (Box Model)** e técnicas de layout **Float** e **Inline-Block** para criar uma seção de *Review de Produto* de nível intermediário.

## 🎯 Objetivo do Exercício

O principal objetivo é demonstrar o domínio sobre as propriedades fundamentais do CSS que governam o layout e o espaçamento dos elementos HTML:

1.  **Modelo de Caixas:** Aplicação correta de `padding`, `margin`, `border` e `width`.
2.  **Técnicas de Layout:** Utilização de `display: inline-block;` e `float: left;` para alinhamento horizontal.
3.  **Controle de Fluxo:** Uso de `clear: both;` e `overflow: hidden;` (técnica clearfix) para gerenciar elementos flutuantes.
4.  **Box-Sizing:** Comparação e demonstração das diferenças entre `box-sizing: border-box;` (moderno) e `box-sizing: content-box;` (padrão antigo).
